FAQs


O que é o EDUREC?
Uma ferramenta interativa de visualização da rede municipal de ensino da cidade do Recife.

O que é o IDEB?
IDEB é o Índice de Desenvolvimento da Educação Básica, criado em 2007, pelo Instituto Nacional de Estudos e Pesquisas Educacionais Anísio Teixeira (INEP). Foi formulado para medir a qualidade de aprendizado nacional e estabelecer metas para a melhoria do ensino.

O que é Correção de Fluxo?
Criado pelo Ministério da Educação, a Política de Correção de Fluxo Escola foi uma solução para o problema da defasagem idade-ano escolar. Consiste em processos de ensino e aprendizagem que se proprõem, em um ano letivo, desbloquear emoções negativas dos alunos em relação a escola.

O que são os ciclos de ensino especiais?
São ciclos de ensino voltados para indivíduos com necessidades especiais.

O que é o Programa de Educação para Jovens e Adultos?
É uma modalidade de ensino voltada para pessoas que não concluíram o ensino fundamental e/ou médio na idade apropriada.

Sobre


O projeto visa prover uma ferramenta, que faz uso de diferentes técnicas de visualização de dados, para analisar o conjunto dos alunos e das escolas da rede municipal do Recife, dados disponíveis em: Dados abertos Recife.
